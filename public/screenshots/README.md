# Screenshots

Place your screenshot images here for the README:

- `hero.png` — Landing page / hero section
- `create-goal.png` — Goal creation page
- `dashboard.png` — User dashboard
- `community.png` — Community feed

## How to Capture

1. Run the app locally: `npm run dev`
2. Open Chrome DevTools → Device Toolbar → select a laptop viewport (1280×800)
3. Navigate to each page and take a screenshot (Cmd+Shift+P → "Capture screenshot")
4. Save screenshots to this directory with the filenames listed above
